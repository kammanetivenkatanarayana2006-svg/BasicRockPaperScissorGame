# 🪨📄✂️ Basic Rock Paper Scissor Game

A simple Rock Paper Scissor game built using HTML, CSS, and JavaScript.

## Features

* Play Rock, Paper, Scissor against the computer
* Random computer moves
* Instant result display
* Score tracking
* Simple and responsive design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

```text
index.html
style.css
script.js
README.md
```

## 🎮 How to Play

1. Click Rock, Paper, or Scissor.
2. The computer will randomly choose its move.
3. The winner will be displayed instantly.
